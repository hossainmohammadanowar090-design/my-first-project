# Responsive Website Design

A clean and responsive website built using HTML and CSS.

## Features
- Uses relative paths (no local file paths)
- Fully responsive layout
- Stable layout on zoom in/out
- Clean navigation and banner section
- Error-free loading

## Technologies Used
- HTML5
- CSS3
- jQuery
- FlexSlider

## Project Structure
- Organized folders for CSS, JS, and images
